# 📊 CSV Data Analyzer (GUI)

## 📌 Description

A simple GUI-based application to analyze CSV data by calculating average, maximum, and minimum values.

## 🚀 Features

- Load CSV file
- Display data
- Calculate average, max, min
- Simple GUI interface

## 🛠 Technologies

- Python
- Tkinter
- CSV module

## ▶️ How to Run

```bash
python csv_gui.py
```

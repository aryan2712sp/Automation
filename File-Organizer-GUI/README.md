# 📂 File Organizer GUI (Python)

## 📌 Description

A GUI-based Python application that automatically organizes files into folders based on file types.

## 🚀 Features

- Select folder using GUI
- Automatically categorize files (Images, PDF, Music, etc.)
- Simple and user-friendly interface

## 🛠 Technologies

- Python
- Tkinter (GUI)
- os, shutil

## ▶️ How to Run

```bash
python organizer_gui.py
```
